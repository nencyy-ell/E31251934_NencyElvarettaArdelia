Proyek ini dibuat untuk memenuhi tugas mata kuliah Pemrograman Berorientasi Objek (PBO). Program ini mendemonstrasikan penerapan konsep Polimorfisme dan Method Overriding melalui studi kasus perhitungan luas dan keliling pada berbagai bentuk bangun datar.

Deskripsi Tugas

Program ini terdiri dari satu superclass (induk) dan tiga subclass (anak) yang menerapkan konsep pewarisan (inheritance):

1. BangunDatar.java: Kelas induk yang mendefinisikan metode dasar luas() dan keliling().
2. Persegi.java: Implementasi perhitungan luas dan keliling persegi (Sisi x Sisi).
3. Lingkaran.java: Implementasi perhitungan luas dan keliling lingkaran menggunakan konstanta Math.PI.
4. Segitiga.java: Implementasi perhitungan luas segitiga (Alas x Tinggi / 2).
5. Main.java: Kelas utama yang berfungsi untuk menjalankan program dan menampilkan hasil eksekusi pada terminal.
